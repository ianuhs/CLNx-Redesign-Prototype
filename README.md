# CLNx Job Flow Prototype

This is a PyGame prototype that simulates a redesigned on-campus job journey:

- Login page
- On-campus job search page
- Filter panel with functional dropdowns
- Structured job details page
- Application upload page

## Run

1. Create and activate a virtual environment if you want one.
2. Install dependencies:

```bash
python3 -m pip install -r requirements.txt
```

3. Start the prototype:

```bash
python3 main.py
```

## Notes

- The interface is intentionally a desktop-window simulation, not a browser site.
- Job data is placeholder content shaped to match the requested structure.
- Upload controls are visual only, per the prototype brief.
